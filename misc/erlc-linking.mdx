---
title: "Linking your ER:LC Server to Elevate"
---

<Warning>
  You are required to be the **owner** of the ER:LC private server in order to be able to access the server settings which are required for this. Your server must also have the **API pack** enabled.
</Warning>

<Steps>
  <Step title="Link your server">
    1. Join your ER:LC server
    2. Click the **Server Info** button in the top right corner of the screen
    3. Click **Edit Server Settings**
    4. Search for **API Key**
    5. Press **Edit** and copy the key
    6. Run `/setup`  with Elevate and go to the **ERLC Integration Module**
    7. Select **Set API Key** and paste in the key
    8. Press **Submit** and now your server is linked!
  </Step>
</Steps>
